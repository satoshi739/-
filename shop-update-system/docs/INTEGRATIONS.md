# 連携チャネル（整理用）

実装順は **マスタ固定 → テキスト出力 → API**。権限が取れたチャネルから足す。

## 一覧

| チャネル | v1 | 備考 |
|----------|-----|------|
| 社内マスタ（YAML） | 必須 | 単一の正 |
| Google Business Profile | 手動コピペ or 将来API | ロケーション・OAuth |
| Instagram / Facebook | 手動 or 将来 Graph API | ビジネスアカウント・権限 |
| WordPress | 手動 or REST API | アプリパスワード・エンドポイント |
| チラシ・店頭POP | 手動 | 出力はテキスト＋指示 |
| 予約サイト（各種） | 手動 | URL・文言はマスタの `links` |

## API を触るときの共通メモ

- レート制限・二重投稿防止（冪等性）
- テスト用アカウントで先に検証
- 失敗時のロールバック（マスタを戻す手順）
