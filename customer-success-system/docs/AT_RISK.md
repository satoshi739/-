# 離反（アットリスク）の兆候と初動

## 兆候チェックリスト

- [ ] 連絡への返信が遅い・短い・テンションが下がった
- [ ] ミーティングのキャンセル・延期が続く
- [ ] 「効果が感じられない」「期待と違う」などの言質
- [ ] 請求・支払いに関するトラブル（未入金・金額異議）
- [ ] 担当者が変わった（引き継ぎ不足）
- [ ] 競合の名前が出た／見積もりを取っている

## 初動（原則）

1. **事実をメモする**（日付・チャネル・発言の要約）。感情だけで判断しない。
2. **一度、聴く時間を取る**（15〜30分）。「どうすれば続けやすいか」を質問中心で。
3. **スコープと成果物のすり合わせ**を再確認。契約・提案書の範囲を共有する。
4. 値引き・追加無償は**単独決裁せず**、必要なら社内で方針を決めてから提示。

## エスカレーション

- 法的・契約上の争いになりそう → 社内の責任者に即共有
- 未払いが続く → [`finance-system` の請求サイクル](../../finance-system/docs/BILLING_CYCLE.md) の督促段階と連動

## 失注・解約になったら

- `offboarding-checklist.md` を進める
- `project-sheet.yaml` の `phase` を P5 に更新
- [`monthly-finance-log.yaml`](../../finance-system/templates/monthly-finance-log.yaml) の `churned_clients` に記録
