# 納品品質ゲート

## 全案件共通

- [ ] 顧客が承認したスコープの範囲内か
- [ ] 誤字・脱字・表記ゆれがないか
- [ ] UPJグループの `ng_words`（`shop-update-system/` 各 profile.yaml 参照）が含まれていないか
- [ ] 景表法・業法に抵触する表現がないか
- [ ] 数値・実績引用の出典が明記されているか

## DSc Marketing — SNS・コンテンツ系

- [ ] `marketing-system/docs/QUALITY_BAR.md` の基準を満たしているか
- [ ] 投稿文にブランドトーン（`brand_voice`）が守られているか
- [ ] ハッシュタグの種類・数が適切か（過剰禁止）
- [ ] プロフィールリンク・CTA が正しく機能しているか

## DSc Marketing — HP・Web制作系

- [ ] スマートフォン表示で崩れがないか
- [ ] 問い合わせフォームが送信できるか
- [ ] Google Analytics / Search Console の連携が完了しているか
- [ ] ページタイトル・meta description が設定されているか
- [ ] 表示速度：PageSpeed Insights で Mobile ≥ 70

## cashflowsupport — ファクタリング

- [ ] 顧客への条件説明（手数料・リスク）の記録あり
- [ ] 必要書類（請求書・通帳コピー等）が揃っているか
- [ ] 入金確認まで完了しているか

## UPJ — コンサルティング

- [ ] 提案資料に根拠（データ・事例・ロジック）があるか
- [ ] アクションプランに担当者・期日が明記されているか
- [ ] 成果測定の指標（KPI）が合意されているか
