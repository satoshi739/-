# リード流入チャネル管理

## チャネル別の特徴と対応方針

### Instagram（DSc Marketing / UPJ）
- 流入経路：リール投稿 → プロフィール → LINE登録 or DM
- 温度感：低〜中（まず信頼形成が必要）
- 初回返信テンプレ：`templates/instagram-dm-reply.md`
- 注意：「〇〇できますか？」系の問い合わせは即答せず、まずヒアリングへ誘導

### 公式LINE
- 流入経路：各サイト・SNS → LINE登録
- 温度感：中（自発的に登録）
- 対応：登録後24時間以内にウェルカムメッセージ
- ステップ配信：登録→1日後・3日後・7日後（設定済みの場合）

### 問い合わせフォーム（Web）
- 流入経路：各サイト（upjapan.co.jp / dsc-marketing.com / cashflowsupport.jp）
- 温度感：高（具体的な相談意欲あり）
- 対応：24時間以内に電話またはメール返信
- cashflowsupportは特に急ぎの可能性 → **当日対応推奨**

### 紹介（リファラル）
- 流入経路：既存顧客・パートナー
- 温度感：最高（信頼担保済み）
- 対応：紹介者へのお礼連絡を忘れずに

### セミナー・イベント
- 流入経路：勉強会・展示会・コラボ企画
- 温度感：中
- 対応：名刺交換後24時間以内にフォロー

## 月次チャネル別リード数の記録場所

→ `templates/monthly-lead-log.yaml`（月初に新規作成）
