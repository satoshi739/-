# 営業パイプライン

## ステージ定義

```
[L1 リード] → [L2 コンタクト済] → [L3 商談] → [L4 提案済] → [L5 契約] → [L6 失注]
```

### L1 リード（未コンタクト）
- 入口：Instagram DM / 公式LINE / 問い合わせフォーム / 紹介
- アクション：24時間以内に初回返信
- 記録：`templates/lead-sheet.yaml` に起票

### L2 コンタクト済（ヒアリング前）
- 入口：初回返信後、日程調整中
- アクション：ヒアリングシートを事前送付（任意）
- 期限：3営業日以内に商談日確定

### L3 商談（ヒアリング中）
- 形式：オンライン（Zoom/Meet）または対面
- 時間：60〜90分
- 構成：
  1. 自己紹介・アイスブレーク（5分）
  2. 相手の現状・課題ヒアリング（30分）
  3. UPJの解決事例の簡単な紹介（15分）
  4. 提案の方向性合意（10分）
  5. 次のステップ確認（5分）
- 記録：商談メモをlead-sheetに追記

### L4 提案済（見積送付後）
- 成果物：`templates/proposal-template.md` に沿って作成
- 送付後：3営業日でフォローアップ連絡
- 判断基準：2週間返答なし → L6（失注）に移動＋理由メモ

### L5 契約
- 手順：`templates/contract-checklist.md` を全項目チェック
- 署名：電子契約（DocuSign / クラウドサイン）推奨
- 完了後：`project-system/` に案件起票

### L6 失注
- 理由を必ず記録（予算・タイミング・競合・ニーズ違い・その他）
- 6ヶ月後に再アプローチ可否を確認

## KPI（月次）

| 指標 | 目標 |
|------|------|
| 新規リード数 | 20件/月 |
| 商談設定率 | ≥ 50%（リード→商談） |
| 提案率 | ≥ 70%（商談→提案） |
| 成約率 | ≥ 30%（提案→契約） |
| 平均リードタイム | ≤ 14日（リード→契約） |
