# よくある断り文句と返し方

## 原則
- 切り返しは「説得」ではなく「理解の深掘り」
- 断られた理由をメモし、次の提案に活かす
- 無理に押さない。関係を維持して将来の商談に繋げる

---

## DSc Marketing（月額マーケ支援）

### 「今は予算がない」
→「承知しました。予算が確保できるタイミングはいつごろになりそうでしょうか？今後のご参考に、今の費用対効果で一番課題になっている部分を教えてもらえますか？」
→ 低単価プランを提示するのではなく、**優先度の確認**から入る

### 「自社でやっています」
→「そうなんですね。どのくらいの頻度で更新されていますか？一番大変に感じている作業はどのあたりですか？」
→ 否定せず、課題を引き出す

### 「効果が出るか不安」
→「その不安はもっともです。具体的にどういった成果を期待されていますか？何ヶ月で何が変わると感じてもらえますか？」
→ 期待値をすり合わせ、測定可能な指標を一緒に決める

### 「他社と比較中」
→「ぜひ比べてください。比較されるとき、一番重視されるポイントは何ですか？」
→ 競合を否定しない。自社の強みを「導線の設計力」「自社運用できる形にする」で差別化

## cashflowsupport（ファクタリング）

### 「手数料が高い」
→「おっしゃる通り、コストは大切ですね。ちなみに、今の資金繰りの状況で、もし調達が遅れた場合のリスクはどのくらいですか？」
→ コスト vs リスク（機会損失・信用毀損）の比較に誘導

### 「取引先に知られたくない」
→「2社間ファクタリングであれば取引先への通知は不要です。詳しくご説明できますか？」
→ 2社間の選択肢を提示

### 「怪しい・違法では？」
→「ご心配はもっともです。ファクタリングは民法上の債権譲渡で適法なサービスです。契約前に条件を全てご説明しますので、疑問点はどんな小さなことでも確認ください。」

## UPJ コンサルティング

### 「コンサルって何をしてくれるの？」
→「おっしゃる通り、抽象的ですよね。一番お聞きしたいのは、今の事業で"うまくいっていない"と感じている具体的な場面はどこですか？そこから一緒に考えます。」

### 「社内でできる」
→「そうですね。社内にリソースがある方が早い場面も多いです。ただ、構造として設計する視点は外から見た方が見えやすいことがあります。一度だけ壁打ち的にお話しする機会をいただけますか？」
