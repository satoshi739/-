# 契約前確認チェックリスト

契約書を送る前に**全項目**チェックすること。

## 顧客情報の確認
- [ ] 会社名・法人番号（法人の場合）が正しい
- [ ] 請求先住所が確定している
- [ ] 担当者名・役職が合っている
- [ ] 連絡先メール・電話が最新

## 契約内容の確認
- [ ] サービス範囲（スコープ）が文書化されている
- [ ] 料金・税込金額が明記されている
- [ ] 支払い条件（期日・方法）が合意済み
- [ ] 開始日が明確
- [ ] 最低契約期間・解約条件が明記されている

## コンプライアンス確認
- [ ] 誇大表現・効果保証の文言がないか確認済み（景表法）
- [ ] ファクタリング案件の場合：条件説明の記録あり
- [ ] 特定商取引法の適用確認済み

## 書類の準備
- [ ] 契約書（電子契約用PDF or クラウドサイン）が準備済み
- [ ] 振込先口座情報を伝える準備が整っている
- [ ] 必要な場合：秘密保持契約（NDA）の締結

## 社内確認
- [ ] 代表または担当マネージャーが内容を確認済み
- [ ] プロジェクト担当者が決まっている
- [ ] キックオフ日程の候補が準備できている

## 契約後に行うこと
- [ ] `project-system/` に案件起票
- [ ] `finance-system/` に請求スケジュール登録
- [ ] 紹介案件の場合：紹介者へお礼連絡
- [ ] lead-sheet.yaml を L5（契約）に更新
