# 営業システム（Sales System）

UPJグループ3ブランドの**リード獲得 → 商談 → 見積 → 契約**までを管理する。

## 担当ブランド

| ブランド | 主力商材 | 客単価目安 |
|----------|----------|------------|
| DSc Marketing | 月額マーケ支援（導線設計・SNS・LINE・Web） | 25,000〜100,800円/月 |
| cashflowsupport | ファクタリング相談・資金繰りコンサル | 案件依存 |
| UPJ（コーポレート） | 事業設計・コンサルティング | 高単価・個別見積 |

## 鉄則

- 「確実に儲かる」「必ず結果が出る」は**絶対に言わない**
- 初回は **聴く:話す = 7:3**。課題を先に引き出す
- 見積は口頭先行禁止。必ずドキュメントで渡す
