# Step 2 — 秒付き台本・テロップ

Step 1 のメモを前提に、**1本のメッセージ**に絞ったリール台本を作る。

## 制約

- 総尺 **15〜60秒** を目安（オーバーする場合は短縮案を併記）
- 冒頭 **3秒** でフック
- テロップは **1行あたり短く**（画面で読める長さ）
- ナレーションとテロップが違う場合は列を分ける

## 出力形式（表）

| パート | 秒 | 画面 | ナレーション | テロップ |
|--------|-----|------|--------------|----------|
| フック | 0–3 | | | |

## 最後に

- **CTA を1つ** に限定して明記
- 引用パートがある場合、テロップに **出典をどう見せるか**（例：「※〇〇より」）を書く
