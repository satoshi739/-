# 品質バー（出荷前チェック）

すべてのリール設計セットは、公開・納品前に以下を満たす。満たせない項目は **「要人間確認」** と明示し、理由を書く。

## 1. 内容・コンプライアンス

- [ ] 断定的な医療・投資・法律の助言になっていない（該当する場合は専門家確認が必要と明記）
- [ ] 誇大表現・最上級の根拠がない主張をしていない
- [ ] 引用がある場合、**出典と範囲**が設計書に書いてある
- [ ] 要約のみの場合、**元テキストの歪曲**がない（比喩でごまかしていない）

## 2. リールとしての成立

- [ ] 冒頭 **3秒以内** に「続きを見たい理由」がある（フック）
- [ ] 1本のメッセージが **1つに絞られている**（複数なら分割案が出ている）
- [ ] CTA が1つに決まっている（保存・フォロー・リンクなど）
- [ ] 想定尺が **15〜60秒** の範囲で現実的（オーバーならカット案あり）

## 3. ブランド・トーン

- [ ] 指定トーンと矛盾する語尾・敬語が混ざっていない
- [ ] 禁止事項に触れていない
- [ ] 競合の固有名を不必要に貶していない

## 4. 再利用・運用

- [ ] 同じフォーマットで翌日も再現できる（変な都合のいい例外がない）
- [ ] キャプションが **コピペ可能な形** で渡せる（絵文字・改行ルールが統一）

## 判定

| 結果 | 意味 |
|------|------|
| 合格 | 上記すべてチェック、または「要人間確認」なし |
| 条件付き | 「要人間確認」が残るが、**何を誰が見るか**が書いてある |
| 不合格 | 公開・納品しない。プロンプトまたは入力を修正して再実行 |
