# 売上トラッキング

## 管理する指標

| 指標 | 説明 | 更新タイミング |
|------|------|---------------|
| MRR（月次経常収益） | 月額契約の合計金額 | 新規契約・解約のたびに |
| 新規MRR | 当月の新規契約分 | 契約締結後 |
| 解約MRR（チャーン） | 当月の解約分 | 解約確定後 |
| 単発売上 | HP制作・コンサルなど非月額 | 入金確認後 |
| 粗利率 | 売上 - 外注費 | 月末集計 |

## MRR の計算例

```
MRR = Σ（各月額顧客の月額料金・税抜）

例：
- 顧客A: 39,800円（税込）= 36,181円（税抜）
- 顧客B: 79,800円（税込）= 72,545円（税抜）
- MRR = 36,181 + 72,545 = 108,726円
```

## 目標MRR（参考）

| フェーズ | 目標MRR | 顧客数目安 |
|----------|---------|-----------|
| 初期 | 300,000円 | 8〜10社 |
| 成長期 | 1,000,000円 | 25〜30社 |
| 安定期 | 3,000,000円 | 50〜60社 |

## 月次レビューで確認すること

1. **MRR増減**：新規vs解約の純増
2. **チャーン率**：解約MRR / 前月末MRR（目標：< 5%/月）
3. **LTV（顧客生涯価値）**：平均月額 × 平均継続月数
4. **回収遅延**：未入金件数・金額
5. **ブランド別売上比率**：DSc / CSF / UPJ
